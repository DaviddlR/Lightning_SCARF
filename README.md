# Network intrusion detection on the UNSW-NB15 dataset


In this project we have tested two different Self-Supervised Learning (SSL) approaches, Contrastive learning and Reconstruction-based SSL, for intrusion detection on the UNSW-NB15 dataset. The code is organized as described below.


## fullTrainingAndEvaluation.py
In this file, we can train and evaluate a SSL model on the UNSW-NB15 dataset.


## supervisedClassifier.py
In this file, we can train and evaluate several supervised models on the UNSW-NB15 dataset.
